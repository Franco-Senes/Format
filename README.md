Format
======

About
-----
This project is called "Format." It's a google chrome extension that tidy's your windows with ai and adds extra features to your browser

How to use
----------
1. Clone the github repository.
2. Open your chromium browser
3. Go to your extension settings 
4. Select the developer mode toggle
5. Select load uncompressed
6. Then select the carpet where the github repo sits in.
7. You now have format!
